# Screenshots Directory

This directory contains screenshots of the HomeBuilderBlinds application.

## Required Screenshots

Please add the following screenshots to this directory:

1. **homepage.png** - Customer-facing homepage
2. **operations-dashboard.png** - Operations dashboard
3. **measurement-scheduling.png** - Measurement scheduling calendar
4. **accounting-dashboard.png** - Accounting/financial dashboard
5. **github-activity.png** - GitHub profile showing commit history

## How to Add Screenshots

1. Save your screenshots with the exact filenames listed above
2. Place them in the `screenshots/` directory
3. Commit and push to GitHub

The README.md references these images, so make sure the filenames match exactly.
